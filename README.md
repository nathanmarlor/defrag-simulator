# 💾 MS-DOS 6.22 Defrag

Pixel-perfect browser recreation of `DEFRAG.EXE` from MS-DOS 6.22 — licensed from Norton SpeedDisk by Symantec.

**Block characters. Box-drawing borders. Blue screen. Norton menu bar. Exactly how you remember it.**
